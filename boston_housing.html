编程练习3：定义衡量标准Â¶

如果不能对模型的训练和测试的表现进行量化地评估，我们就很难衡量模型的好坏。通常我们会定义一些衡量标准，这些标准可以通过对某些误差或者拟合程度的计算来得到。在这个项目中，你将通过运算决定系数 R2 来量化模型的表现。模型的决定系数是回归分析中十分常用的统计信息，经常被当作衡量模型预测能力好坏的标准。

R2的数值范围从0至1，表示目标变量的预测值和实际值之间的相关程度平方的百分比。一个模型的R2 值为0还不如直接用平均值来预测效果好；而一个R2 值为1的模型则可以对目标变量进行完美的预测。从0至1之间的数值，则表示该模型中目标变量中有百分之多少能够用特征来解释。模型也可能出现负值的R2，这种情况下模型所做预测有时会比直接计算目标变量的平均值差很多。

在下方代码的 performance_metric 函数中，你要实现：

  • 使用 sklearn.metrics 中的 r2_score 来计算 y_true 和 y_predict的R2值，作为对其表现的评判。
  • 将他们的表现评分储存到score变量中。

或

  • (可选) 不使用任何外部库，参考决定系数的定义进行计算，这也可以帮助你更好的理解决定系数在什么情况下等于0或等于1。

from sklearn.metrics import r2_score

def performance_metric(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    
    score = r2_score(y_true, y_predict)

    return score

问题 7- 网格搜索（Grid Search）Â¶

什么是网格搜索法？如何用它来优化模型？

问题 7 - 回答:Â¶

  1. 网格搜索法是系统地遍历各种参数组合(人为定义的组合)，进行模型计算及验证，以避免参数选择随意和盲目的一种方法；
  2. Grid Search优化模型，实际只是通过得到当前模型的最佳匹配参数，以达到最好的训练目的和预测输出；
  3. 它得到最优组合的标准，是我们传入的评分函数所计算得出的分值，所以需要指定评价函数。

问题 8 - 回答：Â¶

  1. 就是将训练数据集切分成k份，每次取一份的数据作为测试集，剩下k-1份数据作训练集。充分利用数据集对模型进行训练及验证；
  2. GridSearchCV分三步完成这个过程：
    • 首先生成给定参数各种组合可能的“网格”
    • 然后给特定的算法传入“网格”中的参数一一进行尝试
    • 拟合函数尝试完所有的参数组合后，根据我们传入的评分函数的评价计算，最后得到最佳的参数组合
  3. GridSearchCV的cvresults属性返回给我们如下信息：
    • 候选参数的一系列设置信息：
      'param_kernel'、'param_gamma'、'param_degree'
    • 训练及测试分值结果等：
      'split0_test_score'、 'split1_test_score'、 'mean_test_score'、 'std_test_score'、 'rank_test_score'、 'split0_train_score'、 'split1_train_score'、 'mean_train_score'、 'std_train_score'、
    • 计算时间等：
      'mean_fit_time'、 'std_fit_time'、 'mean_score_time'、 'std_score_time'、
    • 参数本身：
      'params'
  4. 不使用交叉验证，则在模型和数据集选择上有很大的盲目和随机性，结果的优劣难以评估；
    • 如果简单进行8:2拆分，选择最好的参数组合。还是要看数据集的规模，单次分割对于比较大的数据集，也是不会影响训练结果的；
    • 交叉验证避免的是数据集规模较小情况下的训练问题；
    • 交叉验证将数据分组，每次选取一个组合作为测试集（其他为训练集），这样所有的样本都被作为了训练集和测试集，每个样本都被验证一次；过程中结合评分函数的评价，我们就得出最优解。
  5. 交叉验证通过不同组合，测试，几乎能够充分使用全部的数据集和模型验证，然后从中挑选出最佳；
  6. 补充：
    • 默认情况下，kfold对于数据是按顺序进行切分的

    scoring_fnc = make_scorer(performance_metric)

    grid = GridSearchCV(estimator=regressor, param_grid=params, scoring=scoring_fnc, cv=cross_validator)

    # 基于输入数据 [X,y]，进行网格搜索
    grid = grid.fit(X, y)

    # 返回网格搜索后的最优模型
    return grid.best_estimator_

模型健壮性Â¶

一个最优的模型不一定是一个健壮模型。有的时候模型会过于复杂或者过于简单，以致于难以泛化新增添的数据；有的时候模型采用的学习算法并不适用于特定的数据结构；有的时候样本本身可能有太多噪点或样本过少，使得模型无法准确地预测目标变量。这些情况下我们会说模型是欠拟合的。
